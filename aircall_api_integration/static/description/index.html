HOW TO USE

First of all click on the setting icon as shown below:


When the user clicks on the settings menu it will be redirected to the settings page as shown below:
URL: Set Aircall API URL.
API ID: Set Aircall API ID, User will get it from Aircall Account.
API Token: Set Aircall API Token, User will get it from Aircall Account.
Aircall Integration Token: Set Webhook Token, User need to integrate Webhook with aircall and generate a token from there.


- Contacts
Click on the Contacts as shown below:


Once user click on the Create button it will open the form like this:
The user has to fill in the necessary information as shown below in the form.
Where Name, Last Name, Phone(with valid country code), Email are mandatory fields.
Once the contact is created it will automatically created in aircall account after sometime.
Odoo and Aircall both are synced with each other so whatever contact is created in one medium can be visible in both.


- Click to Dial or CTI
When the user clicks on the phone number it will open a dial pad and the number will be set on the dial pad.
Users can click on the call button and dial the call.


- Insights Card
As the call is ongoing it shows contact details in the view details option.
An Insights Card will pop up from downward which shows Odoo's contact name and lead/opportunities.
On click on contact it will redirect to the Odoo contact and
on click Opportunity, if contact has multiple opportunities then it will redirect to the tree view that will show multiple lines
and if contact has only one opportunity for that contact then it will open form view.


- Call Logs
When the call is ongoing. User will get options for adding notes and tags,
so if the user added any note or tags from here then it will be added to Odoo contact in the call log,
if the contact has lead then call logs added on lead as well as contact,
if the user has multiple opportunities then call logs will be added on latest opportunity.


When the user ended the call, after 30 seconds the call log will be available on contact and lead as in below screenshot:
Also it attaches the call recording to the contact which is displayed in the tab named "Aircall Details", if the recording exists for that call.


User can open Aircall from Odoo, by simply clicking on Aircall icon,
User can open Aircall from Odoo, by simply clicking on Aircall icon, user can also dial a call or see the same CTI as Aircall Phone in Odoo.


- Leads in Odoo
Users can see Opportunities from the contact, the button which is available as per the below screenshot will show the numbers of opportunities available in related contact, and click on that button user will be redirected to the Opportunity view.


After Clicking/Creating on Opportunity button it will show as in the below screenshot:


- Webhook Integration with Aircall
Users need to Integrate Webhook with Aircall to use Webhook events for integration with Odoo.
User need to setup Odoo URL in webhook settings for generate a token and use that token in odoo configuration.


Enable Webhook events for Contact creation, Call logs/comments.
Users have to select only webhook events which are shown as per the below image.
The selected webhook events are listed here,
1) contact.created
2) call.created
3) call.ended
4) call.tagged
5) call.untagged
6) call.commented
